安装

运行

安装主题

可以自行去hexo官网找主题

当然我还是推荐新手使用fluid主题

  • 主题下载到kongqi/themes目录下
  • 修改配置文件

上传在github

当然你不上传也可以用

  • 生成ssh秘钥
  • ssh秘钥上传至github
  • 在github创建项目
  • 设置git信息
  • 上传hexo
  • 运行 and 绑定域名

生成ssh密钥

两个中选一个

查看密钥

ssh密钥上传至github

  • 打开浏览器访问github
  • 登录(没有的册一个账号)
  • 点击你的头像
  • 点击Settings
  • 点击SSH and GPG keys
  • 点击New SSH key
  • 输入标题
  • 输入xx.pud的所有内容
  • 点击Add SSH key

测试

如果输出为Hi github用户名! You’ve successfully authenticated, but GitHub does not provide shell access.
则正确

创建仓库

  • 点击头像附近的+号
  • 点击New repository
  • 然后按要求去输入
  • 点击Create repositorv

创建完成了

上传hexo

绑定域名

  • 进入你的仓库
  • 点击Settings
  • 点击Pages
  • 操作
    Branch 那栏中
    要是这样的
    main /root
    如果不是要改成这样 改好后点save
  • 等待几分钟就可以了
    在Pages中看见 github用户名+github.io就成功了

绑定域名

DNS中添加
类型:CNAME 名称:@ 目标:github用户名.github.io
当然你可以再添加一个 只需要把上一步的@ 改成www

在Pages中
Custom domain那栏中
输入你的域名
点击Save
